## Project commands
### Prepare project

```bash
    # Install dependencies
    yarn install
```

### Run project

```bash
    # Run project
    yarn dev
```

### Run test

```bash
    # Run test
    yarn test
```

### Run build

```bash
    # Run build for production
    yarn build
```
